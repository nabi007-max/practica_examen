# Reposteria Flask

Aplicacion web en Flask para la gestion basica de productos, usuarios y ventas, con panel administrativo, autenticacion y soporte de comprobantes PDF.

## Funcionalidades principales

- Inicio de sesion, registro y recuperacion de contrasena.
- Administracion de productos, ventas y detalle de ventas.
- Confirmacion y anulacion de ventas con control de stock.
- Generacion de comprobantes PDF.
